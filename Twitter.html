Twitter
sentiment analysis and stock price movements: Machine learning approach



In this digital age, social media sentiments, especially from Twitter, influence financial markets. This project explored the link between Twitter sentiments and stock dynamics, utilizing various sentiment analysis tools. We further employed machine learning, specifically Linear Regression and Random Forest regression, to predict stock movements based on these sentiments.


Note: The Project is still ongoing and is being improved.
